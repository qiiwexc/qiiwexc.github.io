Installing Windows 11 with a local account
  1. Start Windows 11 installation normally
  2. After the restart, when asked to sing in, press Shift + F10
  3. In the command prompt window type the start ms-chx:localonly command
  4. Close the command prompt window and proceed with creating a local Windows account
Older builds
  1. Start Windows 11 installation normally
  2. After the restart, press Shift + F10
  3. In the command prompt window type the OOBE\BYPASSNRO command to disable the Internet connection requirement (the computer will restart)
  4. After the restart, press Shift + F10 again
  5. Type ipconfig /release command to disable the Internet connection or just unplug the Ethernet cable
  6. Close the command prompt window and proceed with creating a local Windows account
